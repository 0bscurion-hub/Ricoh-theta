# \_HDMIreso

### Overview

Resolution output for HDMI.

The current setting can be acquired by [camera.getOptions](../commands/camera.get_options.md), and it can be changed by [camera.setOptions](../commands/camera.set_options.md).

### Support value

| Value | Description |
|:--|:--|
| Auto | Auto |
| L | 1920 x 1080 |
| M | 1280 x 720 |
| S | 720 x 480 |
