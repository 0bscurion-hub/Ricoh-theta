# 0xD407 PerceivedDeviceType

### Device Prop Code

0xD407

### Overview

Acquire the device type of the camera.  
This property value is fixed to "0x00000001" in RICOH THETA

### Support model

| X | Z1 | V | SC | S |
|:--|:--|:--|:--|:--|
| All | All | All | All | All |

### Support value

| Value | Description |
|:--|:--|
| 0x00000000 | Generic device |
| 0x00000001 | Camera |
