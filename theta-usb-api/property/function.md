# 0xD824 Function

### Device Prop Code

0xD824

### Overview

Acquires and sets the Shooting function.  
(Vendor Extension Property)

### Support model

| X | Z1 | V | SC | S |
|:--|:--|:--|:--|:--|
| All | All | --- | --- | --- |

### Support value

| Value | Description |
|:--|:--|
| 0x00 | Normal shooting function |
| 0x01 | Self-timer shooting function (RICOH THETA X is not supported) |
| 0x02 | My setting shooting function |
