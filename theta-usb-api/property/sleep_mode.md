# 0xD80E SleepMode

### Device Prop Code

0xD80E

### Overview

Acquire or set the camera sleep status.  
(Vendor Extension Property)

### Support model

| X | Z1 | V | SC | S |
|:--|:--|:--|:--|:--|
| All | All | All | All | All |

### Support value

| Value | Description |
|:--|:--|
| 0x00 | Not in sleep mode |
| 0x01 | In sleep mode |
