# 0xD83D ExposureStatus

### Device Prop Code

0xD83D

### Overview

Acquires the exposure status of the camera.  
(Vendor Extension Property)  

### Support model

| X | Z1 | V | SC | S |
|:--|:--|:--|:--|:--|
| -- | v3.10.2 or later | -- | -- | -- |

### Support value

| Value | Description |
|:--|:--|
| 0x00 | Not exposured, Ready to capture. |
| 0x01 | During exposured. |
| 0x02 | During image processing. |
