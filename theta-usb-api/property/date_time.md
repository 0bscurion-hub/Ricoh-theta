# 0x5011 DateTime

### Device Prop Code

0x5011

### Overview

Acquire or set the date and time.

### Support model

| X | Z1 | V | SC | S |
|:--|:--|:--|:--|:--|
| All | All | All | All | All |

### Support value

The supported format is "YYYYMMDD" + "T" + "hhmmss".  
Time zone can be set by appending the format "Z" or "Â±hhmm".

When decimals are set, round off all numbers after the decimal point to set the time.
