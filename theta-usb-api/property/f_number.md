# 0x5007 F-Number

### Device Prop Code

0x5007

### Overview

Acquire or set the aperture value.

### Support value

| X | Z1 | V | SC | S |
|:--|:--|:--|:--|:--|
| --- | All | --- | --- | --- |

### Support value

#### When the exposure program ([ExposureProgramMode](exposure_program_mode.md)) is set to Manual or Aperture Priority

2.1, 3.5, 5.6

#### Otherwise

0 (AUTO)
