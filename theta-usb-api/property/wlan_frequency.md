# 0xD821 Wlan Frequency

### Device Prop Code

0xD821

### Overview

Acquires or sets the woreless LAN frequency.  
(Vendor Extension Property)

### Support model

| X | Z1 | V | SC | S |
|:--|:--|:--|:--|:--|
| All | All | All | --- | --- |

### Support value

| Value | Description |
|:--|:--|
| 0 | 2.4 GHz |
| 1 | 5 GHz (in the case when 5 GHz Wlan is available) |
