# 0xD80F CompositeShootingTime

### Device Prop Code

0xD80F

### Overview

Acquires or sets the shooting time (sec) for interval composite shooting.  
(Vendor Extension Property)

### Support model

| X | Z1 | V | SC | S |
|:--|:--|:--|:--|:--|
| --- | All | --- | v1.10 or later | v01.82 or later |

### Support value

600 to 86400. In 600-second units.
