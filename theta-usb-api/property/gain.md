# 0xD81F Gain

### Device Prop Code

0xD81F

### Overview

Acquires or sets the microphone gain.  
(Vendor Extension Property)

### Support model

| X | Z1 | V | SC | S |
|:--|:--|:--|:--|:--|
| All | All | All | --- | --- |

### Support value

| Value | Description |
|:--|:--|
| 0x00 | Normal mode |
| 0x01 | Loud volume mode |
| 0x02 | Mute mode<br>(RICOH THETA V firmware v2.50.1 or later, RICOH THETA X is not supported) |
