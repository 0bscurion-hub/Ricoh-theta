# 0xD814 BluetoothPowerStatus

### Device Prop Code

0xD814

### Overview

Acquires or sets the power ON/OFF status of the Bluetooth module.  
(Vendor Extension Property)

### Support model

| X | Z1 | V | SC | S |
|:--|:--|:--|:--|:--|
| All | All | All | --- | --- |

### Support value

| Value | Description |
|:--|:--|
| 0 | Bluetooth and BLE is disabled (OFF). |
| 1 | Bluetooth and BLE is enabled (ON). |
