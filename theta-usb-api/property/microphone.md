# 0xD81C Microphone

### Device Prop Code

0xD81C

### Overview

Acquires or sets the microphone to be used.  
(Vendor Extension Property)

### Support model

| X | Z1 | V | SC | S |
|:--|:--|:--|:--|:--|
| --- | All | All | --- | --- |

### Support value

| Value | Description |
|:--|:--|
| 0x00 | AUTO |
| 0x01 | Internal microphone |
| 0x02 | External microphone (RICOH THETA V) |
