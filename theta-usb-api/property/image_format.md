# 0xD827 ImageFormat

### Device Prop Code

0xD827

### Overview

Acquires or sets the format for still image.  
(Vendor Extension Property)

### Support model

| X | Z1 | V | SC | S |
|:--|:--|:--|:--|:--|
| All | All | --- | --- | --- |

### Support value

| Value | Description |
|:--|:--|
| 0x00 | JPEG |
| 0x01 | RAW+ (RICOH THETA X is not supported) |
