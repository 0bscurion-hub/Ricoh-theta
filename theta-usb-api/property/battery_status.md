# 0xD80C BatteryStatus

### Device Prop Code

0xD80C

### Overview

Acquire the camera charging status.  
(Vendor Extension Property)

### Support model

| X | Z1 | V | SC | S |
|:--|:--|:--|:--|:--|
| All | All | All | All | All |

### Support value

| Value | Description |
|:--|:--|
| 0x00 | Not charging status |
| 0x01 | During charge |
| 0x02 | Fully charged |
