# 0xD802 AutoPowerOffDelay

### Device Prop Code

0xD802

### Overview

Acquires or sets the auto power off time (min) of the camera.   
(Vendor Extension Property)

### Support model

| X | Z1 | V | SC | S |
|:--|:--|:--|:--|:--|
| --- | --- | --- | All | All |

### Support value

0 (no auto power off), 1 to 30
