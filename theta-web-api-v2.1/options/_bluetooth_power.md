# \_bluetoothPower

### Overview

Power ON/OFF status of the Bluetooth module.

Can be acquired by [camera.getOptions](../commands/camera.get_options.md) and set by [camera.setOptions](../commands/camera.set_options.md).

### Support model

| X | Z1 | V | SC | S |
|:--|:--|:--|:--|:--|
| All | All | All | --- | --- |

### Support value

| Value | Description |
|:--|:--|
| ON | Power ON status |
| OFF | Power OFF status |
