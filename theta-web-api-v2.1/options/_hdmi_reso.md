# \_HDMIreso

### Overview

Resolution output via HDMI.

Can be acquired by [camera.getOptions](../commands/camera.get_options.md) and set by [camera.setOptions](../commands/camera.set_options.md).

### Support model

| X | Z1 | V | SC | S |
|:--|:--|:--|:--|:--|
| --- | --- | --- | All | All |

### Support value

| Value | Description |
|:--|:--|
| Auto | Automatic |
| L | 1920x1080 |
| M | 1280x720 |
| S | 720x480 |
