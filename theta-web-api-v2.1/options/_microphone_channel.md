# \_microphoneChannel

### Overview

Microphone channel.

Can be acquired by [camera.getOptions](../commands/camera.get_options.md) and set by [camera.setOptions](../commands/camera.set_options.md).

### Support model

| X | Z1 | V | SC | S |
|:--|:--|:--|:--|:--|
| --- | All | All | --- | --- |

### Support value

| Key | Description |
|:--|:--|
| 4ch+1ch | 360 degree spatial audio + monaural |
| 1ch | Monaural |
