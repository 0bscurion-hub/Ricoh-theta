# \_language

### Overview

Language used in camera OS.

Can be acquired by [camera.getOptions](../commands/camera.get_options.md) and set by [camera.setOptions](../commands/camera.set_options.md).

### Support model

| X | Z1 | V | SC | S |
|:--|:--|:--|:--|:--|
| All | All | All | --- | --- |

### Support value

"en-US", "en-GB", "ja", "fr", "de", "zh-TW", "zh-CN", "it", "ko"
