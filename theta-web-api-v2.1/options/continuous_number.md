# continuousNumber

### Overview

Number of shots for continuous shooting.

The current setting can be acquired by [camera.getOptions](../commands/camera.get_options.md).

### Support model

| X | Z1 | V | SC | S |
|:--|:--|:--|:--|:--|
| All | --- | --- | --- | --- |

### Support value

| Value | Description |
|:--|:--|
| 8 | Maximum value (11K) |
| 20 | Maximum value (5.5K) |
