# Wlan Frequency

### Service

F37F568F-9071-445D-A938-5441F2E82399

### Characteristic

C4B7DFC0-80FD-4223-B132-B7D25A59CF85

### Format

sint8

### Overview

Acquires or sets the wireless LAN frequency of the camera.

### Value Fields

| Value | Description |
|:--|:--|
| 0 | 2.4 GHz |
| 1 | 5 GHz (in the case when 5 GHz Wlan is available) |

### Properties

| Property | Requirement |
|:--|:--|
| Read | Mandatory |
| Write | Mandatory |
| Notify | Excluded |
